#9 Critiquing the Student Information System (SIS)

Academic Progress

The Academic Progress menu effectively shows where a student

is at in their academic career. It shows all the requirements

to graduate and highlights the courses students have and still

need to complete. The features included makes the overall

student experince better and is easy to find.

 

Advisor Tab

The advisor tab on the other hand, is very difficult to find

requiring students to navigate through several different menus

just to get help from an advisor. I think this tab should

be moved to the main menu where students can utilize this

essential tool becuase it already has a lot going for it

when it can be found.

 

Registering for Classes

  • Scheduling
  • Shopping Cart

Scheduling for classes seems overall very intuitive on sis.

Classes are easy to find and register for and a schedule

is provided to show the courses you have once they are added.

However, this schedule requires another tab to access and

does not really allow options to build a different schedule

without first removing courses. I feel like a schedule builder

tool may solve a lot of these downsides while maintaining what

already works very well.

 

When adding courses to your schedule, they can be added

to what is known as the shopping cart. The shopping cart

is a tool used to store classes for later use. While the

feature does work for it's intended purpose, much like the

website itself, it leaves a lot to be expected. In the case of the

shopping cart, the naming does not make a lot of sense and

comes off like it was taken from another website. I believe that

the shopping cart and SIS as a whole could use this a step to

revamp the focus of the site.
